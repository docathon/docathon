readtext

  • Documentation https://kbenoit.github.io/quanteda/
  • Github organization https://github.org/kbenoit/readtext
  • Docathon project https://github.com/kbenoit/quanteda/projects/3
  • Description readtext is a one-function companion package for quanteda, but also one that works completely generally, to do exactly what it says on the tin: It reads files containing text, along with any associated document-level metadata, which we call "docvars", for document variables. Plain text files do not have docvars, but other forms such as .csv, .tab, .xml, and .json files usually do. It can also accept wildcard characters.
